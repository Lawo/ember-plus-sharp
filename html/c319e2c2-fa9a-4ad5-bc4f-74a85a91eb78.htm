Page 3: Dynamic Interface

The dynamic interface can be used to connect to a provider with a database containing unknown elements. It is best suited for applications that are able to consume a database of just about any contents, like e.g. Ember+ Viewer. As we will see, the dynamic interface is poorly suited for applications that have certain expectations about the contents of the database.
