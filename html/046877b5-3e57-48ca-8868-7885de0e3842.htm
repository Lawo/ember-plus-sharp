Welcome to the Ember+ Sharp Project

The primary goal of this project is to make Ember+ as simple as possible for .NET developers. If you are unfamiliar with Ember+, it is recommended to read at least the Introduction of the Ember+ Specification[1] (currently pages 8 and 9).

This topic contains the following sections:

  • Ember+ Sharp Library
  • Ember+ Sharp Tools
  • Releases
  • Known Issues / Feedback
  • Future Directions
Ember+ Sharp Library

The Ember+ Sharp Library provides the means to quickly implement fully type-safe Ember+ consumers in just about any .NET language.

Ember+ Sharp Tools

  • Glow Analyzer Proxy is a Windows application that acts as a middleman between an Ember+ consumer and a provider that logs and visualizes all traffic.

  • Convert Glow Log is a Console application that converts XML log files as they have been written by Glow Analyzer Proxy into an even more human-readable format.
