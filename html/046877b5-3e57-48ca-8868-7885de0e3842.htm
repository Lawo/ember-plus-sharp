Welcome to the Ember+ Sharp Project

[This is preliminary documentation and is subject to change.]

The primary goal of this project is to make Ember+[1] as simple as possible for .NET developers. For this purpose the following is provided:

  • Ember+ Sharp Library

  • Tools that help capture, visualize and analyze Ember+ communication between a consumer and a provider.

Ember+ Sharp Library

This library implements most parts of the protocol described in the Ember+ Specification[1] . If you are unfamiliar with Ember+, it is recommended to read the Introduction (currently pages 8 and 9).

First and foremost, this library provides the means to quickly implement an Ember+ consumer, which can then be used in an application that needs to communicate with an existing Ember+ provider. Most of the types supporting this use case are demonstrated by the Lawo.EmberPlus.Model Namespace Tutorial.

As a byproduct, this library also offers support for lower level Ember+ communication, as demonstrated by the Lawo.EmberPlus.S101 Namespace Tutorial and the Lawo.EmberPlus.Ember Namespace Tutorial.

Comprehensive Reference documentation is available for all types.

Ember+ Sharp Tools

  • GlowAnalyzerProxy is a Windows application that acts as a middleman between an Ember+ consumer and a provider that logs and visualizes all traffic.

  • ConvertGlowLog is a Console application that converts XML log files as they have been written by GlowAnalyzerProxy into an even more human-readable format.
