GlowAnalyzerProxy

GlowAnalyzerProxy is a Windows application that acts as a middleman between an Ember+ consumer and a provider. The application works as follows:

  • Traffic received from either party is immediately forwarded to the other one.

  • All messages are converted into human-readable XML and written into a log file.

  • The main properties of each message are displayed in a scrollable list.

  • The full detail is shown for the currently selected message.

ConvertGlowLog

ConvertGlowLog is a Console application that converts XML log files as they have been written by GlowAnalyzerProxy into an even more human-readable format.
