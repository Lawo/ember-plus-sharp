Rationale
Ember+ Sharp Library

First and foremost, this library provides the means to quickly implement an Ember+ consumer, which can then be used in an application that needs to communicate with an existing Ember+ provider. Most of the types supporting this use case are demonstrated by the Lawo.EmberPlusSharp.Model Namespace Tutorial.

As a byproduct, this library also offers support for lower level Ember+ communication, as demonstrated by the Lawo.EmberPlusSharp.S101 Namespace Tutorial and the Lawo.EmberPlusSharp.Ember Namespace Tutorial.

Comprehensive Reference documentation is available for all types.
