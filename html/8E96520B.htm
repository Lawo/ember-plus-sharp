GlowLogConverter
Converts the log written by an S101Logger instantiated with Instance into an equivalent log the payloads of which can be much more easily read by humans.
GlowLogInterpreter
Reads the events written by a S101Logger instantiated with Instance and applies them to the tree rooted in Root.
GlowTypes
Provides a singleton EmberTypeBag instance containing all Glow types.
