Remarks

This method returns when initial values have been received for all non-optional TRoot properties and recursively for all non-optional properties of FieldNodeTMostDerived subclass objects. Afterwards, all changes are continuously synchronized such that the state of the object tree accessible through the Root property mirrors the state of the tree held by the provider.

All changes to the object tree are reported by raising the PropertyChanged event of the affected objects.
